## 📡 Installation & Deployment

### Prerequisites
- Node.js (v18+)
- MySQL Database
- Google Gemini API Key

### Setup Sequence
1. **Clone the Command Center**:
   ```bash
   git clone https://github.com/Adidam-Akshay-Bhaskar/Fitness-AI.git
   cd Fitness-AI
   ```

2. **Initialize Dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env` file in the `backend/` directory:
   ```env
   PORT=3000
   DB_HOST=your_host
   DB_USER=your_user
   DB_PASSWORD=your_password
   DB_NAME=fitness_tracker
   JWT_SECRET=your_ultra_secure_secret
   GEMINI_API_KEY=your_google_ai_key
   ```

4. **Initialize Database**:
   Import the schema from `backend/db/database-complete-updated.sql` into your MySQL instance.

5. **Engage System**:
   ```bash
   npm run dev
   ```
